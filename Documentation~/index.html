GAOS Event System

A powerful, type-safe event system for Unity with async support and cancellation. Read the development story to learn about the journey and philosophy behind this package.

Introduction

The GAOS Event System provides a robust event management solution built specifically for Unity. It seamlessly integrates with Unity's component lifecycle while delivering type-safe event handling, async support, and flexible event configurations—all with an intuitive API that's easy to use and maintain.

Type-Safe Events

  • Compile-time type checking
  • Interface-based event data
  • Strongly-typed event handlers

Async Support

  • Asynchronous event handling
  • Progress tracking
  • Cancellation support

Developer Experience

  • Priority-based execution
  • Comprehensive error handling
  • ServiceLocator integration
